JavaScript Date 对象以及标准时间格式

1 ISO 8601标准

很多语言都有实现ISO 8601标准。所以我们有必要了解一下ISO 8601标准。ISO 8601标准，全称是《数据存储和交换形式·信息交换·日期和时间的表示方法》。该标准已经到了第三版“ISO8601:2004”，用以替代第一版“ISO8601:1988”和第二版“ISO8601:2000”。
